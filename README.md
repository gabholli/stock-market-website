# Project Title
Stock Market App

## Description
I built this stock market app using the following:

- React Router 7
- Tailwind CSS
- Node.js
- Express.js

## Getting Started
### Dependencies
- Runs within one's web browser of choice
### Installing
- This web page does not require any direct installation on your computer
### Executing program
- To execute, navigate within your web browser to https://stock-market.netlify.app
